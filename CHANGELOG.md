### Added
- Configurable segment suffix stripping via `stripSegmentSuffixes`.

- Default stripped suffixes now cover common Angular + Storybook names:
  `component`, `directive`, `service`, `pipe`, `module`, `guard`, `resolver`, `interceptor`, `stories`, `story`.
- More examples in the hosted storybook instance
