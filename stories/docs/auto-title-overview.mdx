# Storybook Auto Titles

Human-readable Storybook sidebar titles generated directly from your file structure.

This demo Storybook is configured with `autoTitleIndexer()` so the examples in the sidebar show the transformed result instead of Storybook's default raw path output.

```ts
const config: StorybookConfig = {
  experimental_indexers: autoTitleIndexer(),
};
```

Repository:
[github.com/deniszholob/storybook-auto-title](https://github.com/deniszholob/storybook-auto-title)

## What to look at

- `Examples/Ui/Button`
- `Examples/Main Menu`
- `Examples/Utils/Mdx Example`

These examples show path flattening, suffix handling, nested folder preservation, and attached MDX docs.

## Config used in this demo

Check `.storybook/main.ts` in the repository for the active Storybook setup.
